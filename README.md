# Data Analytics Project

This project covers a full data analytics workflow, divided into two main parts: **Exploratory Data Analysis (EDA)** and **Advanced Analytics**. The goal is to show the entire process from understanding raw data to generating deeper, actionable insights.

## Project Structure

The project is broken down into two core pillars:

### 1. Exploratory Data Analysis (EDA)
This is where we set the foundation and get to know the data.
* **Database Exploration:** Understanding the structure and relationships in the data.
* **Dimensions & Date Exploration:** Breaking down categorical data and time-based trends.
* **Measures Exploration:** Looking at key performance indicators (the "Big Numbers").
* **Ranking & Magnitude:** Identifying top/bottom performers and understanding the overall scale of metrics.

### 2. Advanced Analytics
This phase focuses on taking those initial findings and doing deeper dives to find the "so what?" behind the numbers.
* **Cumulative & Performance Analysis:** Tracking growth and evaluating how things are performing against goals.
* **Change-Over-Time:** Pinpointing trends and patterns.
* **Part-to-Whole:** Looking at compositions and proportions (e.g., market share or category breakdown).
* **Data Segmentation & Reporting:** Grouping users/products for targeted insights and presenting them in clear reports.

## How it works
* `/Data_Sets`: Where the raw data lives.
* `/SQL_Query`: Contains the analysis steps, starting with the EDA phase.
* `/reports`: Summaries and final dashboards showing the results.
* * `/docs`: Contains the Documentation & diagrams.

* **Languages:** Python / SQL
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
* **Tools:** Power BI / Tableau (or whatever tools you are actually using)

1. Clone this repo: `git clone [your-repo-link]`
2. Install the required packages: `pip install -r requirements.txt`
3. Dive into the notebooks starting with the `EDA_Phase.ipynb` file.
